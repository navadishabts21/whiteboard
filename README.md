
# NavDishaBTS Whiteboard (Production v3)

Advanced frontend whiteboard tool with:

- 3D-style DB node
- Decision diamond
- Syntax-highlighted code blocks
- Snap-to-grid
- Floating modern toolbar
- Keyboard shortcuts (Ctrl+S, Ctrl+E)
- Mini-map navigation
- Dark mode
- Auto-save
- Download/Upload .board files
- PNG export
- GitHub Pages ready

## Run

npm install
npm run dev

## Deploy

Replace YOUR_GITHUB_USERNAME in package.json
npm run deploy

## Share Logic

Save → share .board file → upload to reload

No backend required.
